# COVID-19 Global Data

All data taken from [WHO](https://covid19.who.int/table).

## Technologies

- .NET Core 2.2
- ElasticSearch
- NEST

## Usage

Download the CSV.\
You can import the data from [Kibana](https://www.elastic.co/kibana) or [Logstash](https://www.elastic.co/logstash).
